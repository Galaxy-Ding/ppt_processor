# 一键生成发包规范工具

## 项目概述
这是一个自动化工具，用于处理PPT文件并生成符合发包规范的文档。主要功能包括：
- 自动识别PPT文件中的关键信息
- 根据模板生成标准化的发包规范文档
- 提供友好的图形用户界面(GUI)

## 功能特点
1. **批量处理**：支持批量处理多个PPT文件
2. **版本识别**：自动识别PPT文件版本(v3等)
3. **日志记录**：详细的处理日志记录
4. **模板支持**：可配置的输出模板

## 技术栈
- **GUI框架**：PyQt5
- **PPT处理**：python-pptx
- **文档生成**：python-docx
- **配置管理**：YAML

## 安装指南
1. 确保已安装Python 3.8+
2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 使用说明
1. 运行主程序：
```bash
python main.py
```
2. 在界面中选择包含PPT文件的目录
3. 点击"一键生成"按钮
4. 生成的文档将保存在各PPT文件所在目录的result子目录中

## 项目结构
```
.
├── core/                # 核心处理逻辑
│   └── packing_file_发包规范.py
├── ui/                  # 用户界面
│   └── 发包规范_window_ui.py
├── config/              # 配置文件
├── extractors/          # 信息提取模块
├── exporters/           # 文档导出模块
├── examples/            # 示例文件
└── main.py              # 程序入口
```

## 注意事项
1. 确保PPT文件命名中包含版本号(如v3)
2. 生成文档前请确认已配置好模板文件
3. 详细配置请参考config目录下的配置文件
